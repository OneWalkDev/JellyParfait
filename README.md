## 概要 

GUIのyoutubeの音楽再生ソフトです。 

chromeで音楽再生するとタブ邪魔だし重くて 

ノートパソコンのメモリも限られてて 

FOOBAR2000バグっちゃったので作りました
 
## インストール方法  

[Release](https://github.com/yurisi0212/JellyParfait/releases) のsetup.exeをダウンロード後、実行すればインストールすることができます。 

## 対応OS 

Windows 10 , Windows 11
